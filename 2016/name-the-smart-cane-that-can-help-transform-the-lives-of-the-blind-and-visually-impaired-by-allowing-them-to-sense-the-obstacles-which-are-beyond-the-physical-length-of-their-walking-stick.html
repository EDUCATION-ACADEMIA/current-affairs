Name the smart cane, that can help transform the lives of the blind and visually impaired by allowing them to sense the obstacles which are beyond the physical length of their walking stick

(A) NewSmartCane

(B) Smart Cane

(C) ** mySmartCane

(D) Super Smart Cane

EXPLANATIONS BELOW

Concept note-1: -WeWALK’s smart cane uses an ultrasonic sensor to detect obstacles above the waist, and it alerts the user via in-hand vibration when these obstacles are nearby.

Concept note-2: -Different Colors on White Cane An all-white cane means that the user is completely blind and has no usable vision. A white cane with a red bottom means that the user has low, but some usable vision. A white and red striped cane means that the user is totally blind and/or deaf.

Concept note-3: -Basically the smart blind cane functions like ordinary blind canes. The difference is that the Smart Cane is equipped with ultrasonic sensor, water sensor and circuit box can be placed. Also the smart blind cane is designed to be foldable so that it is easy for the user to keep.

Concept note-4: -The SmartCane is an electronic device to assist in obstacle detection for the visually impaired. The SmartCane is an electronic device that fits as a handle on walking sticks or white canes used by visually impaired people.
