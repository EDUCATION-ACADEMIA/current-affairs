The Union Government has launched IVRS system in a bid to tackle the call drop menace.

(A) Ironside Volunteer Rescue Squad

(B) ** Integrated Voice Response System

(C) Inland Valley Recovery Services

(D) Independent Valuation and Risk Services

EXPLANATIONS BELOW

Concept note-1: -NEW DELHI: Government is planning to start soon a toll free number 1955 for taking feedback of telecom subscribers on call drops. “Short code 1955 has been allotted for IVRS system on call drop, ” an official source said.

Concept note-2: -“Short code 1955 has been allotted for Interactive Voice Response System (IVRS) on call drop, ” an official said. Provisioning of this number will be mandatory for telecom operators and should be accessible for local and STD calling.
