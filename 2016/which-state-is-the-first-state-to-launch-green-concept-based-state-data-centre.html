Which state is the first state to launch green-concept based State Data Centre

(A) Madhya Pradesh

(B) Mizoram

(C) ** Himachal Pradesh

(D) Kerala

EXPLANATIONS BELOW

Concept note-1: -Detailed Solution. The correct answer is Himachal Pradesh. State Data Centres (SDC) are multiple data centers set up in different states of India to provide basic IT infrastructure for various e-governance programs. The first Indian state to start the State Data Center (SDC), is Himachal Pradesh.
