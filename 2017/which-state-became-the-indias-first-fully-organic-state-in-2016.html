Which state became the India’s first fully organic state in 2016

(A) Punjab

(B) Goa

(C) Andhra Pradesh

(D) ** Sikkim

EXPLANATIONS BELOW

Concept note-1: -In January 2016, Prime Minister Narendra Modi declared Sikkim as India’s first, fully organic state.

Concept note-2: -Sikkim has just become the world’s first organic state!

Concept note-3: -India’s “100 per cent organic state” Sikkim has won the “Oscar for best policies”, conferred by the Food and Agriculture Organisation for the world’s best policies promoting agroecological and sustainable food systems.

Concept note-4: -Sikkim has been acknowledged as the world’s first organic and crime-free state and was bestowed this recognition by the “World Book of Records London.” This honour emphasises that the northeast state has become the first state in the world to implement a 100 per cent organic policy.
