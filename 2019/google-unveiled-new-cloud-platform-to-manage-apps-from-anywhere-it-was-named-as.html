Google unveiled new Cloud Platform, to manage apps from anywhere. It was named as;

(A) Azure

(B) Midru

(C) Kuber

(D) ** Anthos

EXPLANATIONS BELOW

Concept note-1: -Docker is one of the most popular tools for deploying multiple applications on cloud platforms. Docker is most commonly used for deployments.

Concept note-2: -Google Cloud is a suite of cloud computing services that runs on the same infrastructure that Google uses internally for their own consumer products, such as Google Search, Gmail, and YouTube. The list of available Google Cloud services is long-and it keeps growing.

Concept note-3: -Networking, authentication, and security. After you have built your fleet, Anthos helps you manage traffic, manage authentication and access control, and consistently enforce security and compliance policies across your fleet.

Concept note-4: -The Google Cloud SDK provides the primary programmatic interfaces to Google Cloud Platform. Kf enables you to migrate and run applications from the open-source Cloud Foundry platform into containers in Google Kubernetes Engine and Anthos.
