Name the financial service which launched an ‘Identity Check Express’ for uninterrupted secured online transactions.

(A) PayPal

(B) ** Mastercard

(C) American Express

(D) Capital One

EXPLANATIONS BELOW

Concept note-1: -Mastercard on Tuesday announced the launch of Identity Check Express, a next-generation, mobile-first authentication solution that aims to redefine the e-commerce journey for Indian consumers.

Concept note-2: -Mastercard® ID Check TM is a secure code that helps protect you from fraud when you shop online. This code verifies that it’s you making the transaction. How does it work? You will receive a secure one-time code by text or email.

Concept note-3: -What is Mastercard Identity Check? Mastercard Identity Check (formerly “Mastercard SecureCode”) is an authentication procedure from Mastercard for making online purchases with the highest security requirements. This protects you from unauthorised use of your Swiss Bankers card by third parties.

Concept note-4: -MasterCard Identity Check is a suite of security technology solutions that use EMV 3DS2 authentication standards. It works for online transactions or any other transaction where the card is not physically present.
