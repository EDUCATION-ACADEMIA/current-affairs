From which date users will have to dial 0 before dialing any mobile number?

(A) 1 March 2021

(B) 1 January2021

(C) 31 December 2020

(D) ** 15 January 2021

EXPLANATIONS BELOW

Concept note-1: -New Delhi: All landlines users across India will have to prefix ‘0’ for making calls to mobile phones from January 15.

Concept note-2: -Dialling “0” before the mobile phone number was a practice followed for outstation mobile numbers before the government allowed inter-circle mobile number portability, which effectively meant that there was no trunk dialling when it came to mobile numbers and all numbers were considered local.

Concept note-3: -The 0 prefix is for trunk (long-distance) dialing from within the country.

Concept note-4: -Since 2015, calls from mobile phones to any other mobiles do not need to prefix with a 0. However, calls from landlines to non-local mobile numbers need to be prefixed with 0.

Concept note-5: -An integer literal that starts with 0 is an octal number, much like a number starting with 0x is a hexadecimal number.
