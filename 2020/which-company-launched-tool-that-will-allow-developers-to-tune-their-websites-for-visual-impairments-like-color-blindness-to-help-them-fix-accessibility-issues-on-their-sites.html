Which company launched tool that will allow developers to tune their websites for visual impairments like color blindness to help them fix accessibility issues on their sites?

(A) TCS

(B) Microsoft

(C) ** Google

(D) IBM

EXPLANATIONS BELOW

Concept note-1: -Google has launched a new Chrome tool that allows developers to mimic visual impairments like color blindness to help them fix accessibility issues on their sites. Developers can use this feature by launching Google Chrome and heading to the browser’s developer tools.

Concept note-2: -The site wearecolorblind.com offers plenty of insights for designing with the color blind in mind and has deemed several color combinations unsuitable for these individuals, including Green & Black, Green & Grey, Blue & Grey, Light Green & Yellow, Green & Blue, Blue & Purple, Green & Brown, and Green & Red.

Concept note-3: -Colorblindly is an Chrome extension that helps developers and designers simulate different types of color blindness. There are 8 different types of color blindness to simulate: Blue Cone Monochromacy. Achromatomaly-Monochromacy.

Concept note-4: -Colorblindly. Simulates colorblindness in your web browser. Colorblindly is an extension that helps developers create websites for the people with colorblindness by allowing them to simulate the experience those users have on websites.
