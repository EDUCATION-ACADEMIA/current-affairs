In which state first Rice ATM has to be set up?

(A) Maharashtra

(B) Gujrat

(C) Sikkim

(D) ** Karnataka

EXPLANATIONS BELOW

Concept note-1: -Detailed Solution. The correct answer is Bengaluru. Karnataka’s first rice dispensing machine or ‘rice ATM’ will come up in one of the slum areas in Bengaluru. The initiative is part of the Centre’s pilot project ‘Annapurti’, that is being implemented in close partnership with the UN’s World Food Programme.

Concept note-2: -The correct answer is Gurugram, Haryana. India’s first Grain ATM has been set up as a pilot project in Farrukhnagar, Gurugram. The ATM has been named “Annapurti". These Grain ATMs will address all the complaints regarding the right measurement of ration quantity and waiting time.

Concept note-3: -Haryana’s Deputy Chief Minister Dushyant Chatula recently announced that Gurugram is now officially the coveted spot for India’s first food grain ATM ‘Annapurti’. The purpose of this unique ATM is to make the distribution of grains to government-run ration shops easy and hassle-free.

Concept note-4: -The correct answer is Gurugram. The state government of Haryana has kicked off a first-of-its-kind unique initiative in which a grain ATM has been set up in the state’s Gurugram area.

Concept note-5: -India’s first ‘Grain ATM’ was set up in Gurugram, Haryana in 2021. The machine was installed under the World Food Programme (WFP) and it is called the ‘Automated, Multi Commodity, and Grain Dispensing Machine’.
