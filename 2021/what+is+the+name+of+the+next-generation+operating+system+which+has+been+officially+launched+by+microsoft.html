What is the name of the next-generation operating system which has been officially launched by Microsoft?

(A) Window 12

(B) ** Window 11

(C) Window 13

(D) Window X1

EXPLANATIONS BELOW

Concept note-1: -Windows 11 is the latest major release of Microsoft’s Windows NT operating system, released in October 2021. It is a free upgrade to its predecessor, Windows 10 (2015), and is available for any Windows 10 devices that meet the new Windows 11 system requirements.

Concept note-2: -The latest version is Windows 11. The main competitor of this family is macOS by Apple for personal computers and iPadOS and Android for tablets (c.f. Usage share of operating systems § Market share by category). Windows Server: The operating system for server computers. The latest version is Windows Server 2022.

Concept note-3: -As of 2021, Windows 11 is the latest operating system launched by Microsoft.

Concept note-4: -Windows 11 was officially released on 5 October 2021.

Concept note-5: -Windows 11 brings a brand-new, more Mac-like interface to the OS. It features a clean design with rounded corners and pastel shades. The iconic Start menu also moves to the center of the screen along with the Taskbar. But you can move those back to the left, where they are in Windows 10, if you prefer.
