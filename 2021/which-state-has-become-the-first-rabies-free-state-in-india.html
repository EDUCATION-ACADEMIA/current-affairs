Which state has become the first rabies-free state in India?

(A) ** Goa

(B) Delhi

(C) Kerala

(D) Maharashtra

EXPLANATIONS BELOW

Concept note-1: -The correct answer is Goa. This state has become the first Rabies-free state of India. Hence, Option 4 is correct. According to Chief Minister, Pramod Sawant, the state has not reported a single rabies case in the last three years.

Concept note-2: -UPSC Mains Notes: Goa Chief Minister Pramod Sawant announced that the state has not reported a single rabies case in the last three years. This makes Goa the first rabies-free state in the country.

Concept note-3: -Long story short, Goa became India’s first state to achieve human dog-mediated rabies elimination.
