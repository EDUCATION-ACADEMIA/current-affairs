In which state India’s first display fab will be set up?

(A) ** Telangana

(B) Odisha

(C) Kerala

(D) Chhattisgarh

EXPLANATIONS BELOW

Concept note-1: -Recently Telangana Government has signed an MoU with Bengaluru based Elest to set up India’s first Display Fabrication Unit in Hyderabad. This display fabrication unit will be set up with an investment of Rs 24,000 crore.

Concept note-2: -Founded in 1960 by John Bissell to market the diverse craft traditions of India, Fabindia started out as a company exporting home furnishings. The first Fabindia retail store was opened in Greater Kailash, New Delhi fifteen years later.

It will create a much larger ecosystem of partners, ancillaries, and suppliers.
