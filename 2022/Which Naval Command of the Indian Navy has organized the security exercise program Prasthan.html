Which Naval Command of the Indian Navy has organized the security exercise program “Prasthan”?

(A) Southern Naval Command

(B) Eastern Naval Command

(C) Northern Naval Command

(D) ** Western Naval Command

EXPLANATIONS BELOW

Concept note-1: The Offshore Security Exercise, ‘Prasthan’ in the OffshoreDevelopment Area (ODA) off Kakinada was conducted by Eastern Naval Commandon Wednesday.

Concept note-2: The exercise was named ‘Prasthan’by the Indian Navy. Prasthan is conducted twice a year under the aegis ofHeadquarters, Western Naval Command. The exercise which is conductedevery six months aims to integrate the efforts of all maritime stakeholdersinvolved in offshore defense.

Concept note-3: ’Prasthan’ is conductedtwice a year under the aegis of Headquarters, Western Naval Command. Itaims to assess various measures and protocols conceived to address securitythreats and other contingencies that may arise in offshore oil productionplatforms.

Concept note-4: Indian Navy Conducts An Offshore Security Exercise‘Prasthan’ Off Mumbai Coast.

Concept note-5: The correct answer is Prasthan.An offshore security exercise, code-named ‘Prasthan’ was conducted in the OffshoreDevelopment Area (ODA) off Mumbai on 16 November 2021, under the aegis ofWestern Naval Command. Conducted every six months, it is an important element ofensuring offshore security.
