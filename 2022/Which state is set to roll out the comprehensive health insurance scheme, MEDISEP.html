(A) ** Kerala

(B) Madhya Pradesh

(C) Bihar

Concept note-1: Which state is set to roll out the comprehensive health insurancescheme, ‘MEDISEP’? Notes: Kerala government is set to launch MEDISEP schemefrom July 1, 2022. It is a medical insurance scheme for state government employees,pensioners and their eligible family members.
