Which Naval Command of the Indian Navy has organized the security exercise program “Prasthan”?

(A) Southern Naval Command

(B) Eastern Naval Command

(C) Northern Naval Command

(D) ** Western Naval Command

EXPLANATIONS BELOW

Concept note-1: -WESTERN NAVAL COMMAND CONDUCTS OFFSHORE SECURITY EXERCISE PRASTHAN. An offshore security exercise, ‘Prasthan’ was conducted in the Offshore Development Area (ODA) off Mumbai on 23 March 2022, under the aegis of Headquarters, Western Naval Command.

Concept note-2: -The Offshore Security Exercise, ‘Prasthan’ in the Offshore Development Area (ODA) off Kakinada was conducted by Eastern Naval Command on Wednesday.

Concept note-3: -The exercise was named ‘Prasthan’ by the Indian Navy. Prasthan is conducted twice a year under the aegis of Headquarters, Western Naval Command. The exercise which is conducted every six months aims to integrate the efforts of all maritime stakeholders involved in offshore defense.

Concept note-4: -’Prasthan’ is conducted twice a year under the aegis of Headquarters, Western Naval Command. It aims to assess various measures and protocols conceived to address security threats and other contingencies that may arise in offshore oil production platforms.

Concept note-5: -Which Indian Armed Force conducts the ‘Prasthan’ security exercise? Notes: The Indian Navy conducts the ‘Prasthan’ security exercise every six months aims to integrate the efforts of all maritime stakeholders involved in offshore defence.
