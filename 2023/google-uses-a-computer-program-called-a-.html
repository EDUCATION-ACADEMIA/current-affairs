[SOURCES]
Google uses a computer program called a ____

(A) windows

(B) ** web crawler

(C) website

EXPLANATIONS BELOW

Concept note-1: -"Crawler” (sometimes also called a “robot” or “spider") is a generic term for any program that is used to automatically discover and scan websites by following links from one web page to another. Google’s main crawler is called Googlebot.

Concept note-2: -Googlebot is the generic name for Google’s two types of web crawlers: Googlebot Desktop: a desktop crawler that simulates a user on desktop. Googlebot Smartphone: a mobile crawler that simulates a user on a mobile device.

Concept note-3: -A web crawler, or spider, is a type of bot that is typically operated by search engines like Google and Bing. Their purpose is to index the content of websites all across the Internet so that those websites can appear in search engine results.

Concept note-4: -Google Search is a fully-automated search engine that uses software known as web crawlers that explore the web regularly to find pages to add to our index.

Concept note-5: -The correct answer is Web Spider. Web Crawler is also known as Web Spider.
