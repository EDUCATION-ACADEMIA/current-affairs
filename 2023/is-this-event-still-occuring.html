Is this event still occuring

(A) YES

(B) ** NO

(C) Maybe

(D) I think so

EXPLANATIONS BELOW

Concept note-1: -The probability of an event not happening is 1 minus the probability of the event happening.

Concept note-2: -If the occurrence of one event has no effect on the probability of another event, the events are said to be independent. For such events P (A and B) = P(A).

Concept note-3: -The probability that an event will occur is the fraction of times you expect to see that event in many trials. Probabilities always range between 0 and 1. The odds are defined as the probability that the event will occur divided by the probability that the event will not occur.

Concept note-4: -Impossible and Sure Events If the probability of occurrence of an event is 0, such an event is called an impossible event and if the probability of occurrence of an event is 1, it is called a sure event.
